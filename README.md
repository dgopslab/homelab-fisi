# Homelab FISI

Mein persönliches Homelab und Lerntagebuch zur FISI-Umschulung (Fachinformatiker für Systemintegration).

## Ziel

Ein praktisches Lab aufbauen, in dem ich die Inhalte meiner Umschulung erproben kann: Server, Netzwerke, Virtualisierung, Linux, Windows Server, Active Directory, Monitoring — und später Container und Cloud.

## Geplantes Setup

- **Hypervisor:** Proxmox VE auf dediziertem Mini-PC
- **Firewall / Router:** OPNsense (virtualisiert)
- **Netzwerk:** Managed Switch mit VLAN-Trennung
- **Dienste:** Active Directory (Windows Server 2022), Linux-Dienste (DNS, Web, Monitoring), später Docker

## Struktur

- `00-konzept/` — Planung, Netzwerk- und Hardwarekonzept
- `glossar.md` — Begriffserklärungen in eigenen Worten

Weitere Ordner kommen dazu, sobald die jeweiligen Komponenten konfiguriert werden.
